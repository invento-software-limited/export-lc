# What's New in v1.0.1

### Added
- **Frappe Dependencies:**
  - Added mandatory `[tool.bench.frappe-dependencies]` settings in `pyproject.toml` to declare compatibility with Frappe/ERPNext version-16.

### Changed
- **CI Workflow Configurations:**
  - Configured the linters pipeline to trigger on direct pushes to `develop` and `version-16` branches.
  - Removed strict pull request filter from main linter job.

- **Documentation Updates:**
  - Updated key feature lists across README and product documentation to include Sales Order and Commercial Invoice features.
  - Removed outdated references to in-words currency conversion.
